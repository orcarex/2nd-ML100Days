#### 連續值標準化
##### 1. Z-transform: $ \frac{(x - mean(x))}{std(x)} $
##### 2. Range (0 ~ 1): $ \frac{x - min(x)}{max(x) - min(x)} $
##### 3. Range (-1 ~ 1): $ (\frac{x - min(x)}{max(x) - min(x)} - 0.5) * 2 $
